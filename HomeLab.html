    • Software

    Currently, my virtualization is shared betweeen my storage server running Hexos (TrueNAS Scale) and Proxmox. The Raspberry PI is running a local hostassistant instance which I have gained remote access to though the use of cloudflared zero trust tunneling.

    Security

    I am using Cloudflare Zero Trust Tunnels to remove the need for port forwarding and to help reduce the threat level from using consumer equipment and gain a non-dynamic point of access, using Cloudflare also allows for easier DNS management, and fast and simple domain integration with a large number of domain providers. While this is not a perfect system it works well enough for a dorm room system.
